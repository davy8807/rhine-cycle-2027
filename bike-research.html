1. 概覽與關鍵發現

簡單答案：在 Andermatt 本身購買旅遊單車比較困難——這條村很小（約1,500人），大多數是租賃為主的體育用品店。但是，在琉森（從 Andermatt 搭火車1小時）或蘇黎世（火車2小時）購買就非常可行。在旅程結束後賣車也可以通過瑞士二手平台和一些回購計劃實現。

萊茵河路線（Andermatt → 鹿特丹，約1,200+公里）是一條完善的路線，沿途經過許多有單車店的城鎮，所以你也可以中途購買。

2. Andermatt 單車店（主租賃）

以下是在 Andermatt 或其附近確認營業的商店。它們大多提供租賃——有些可能出售單車，但主要是服務想騎一天或週末的遊客，而不是為長途旅行購買旅遊單車。

5. 旅程結束後賣車

網上平台

平台類型外國人友善備註
tutti.ch分類廣告（類似 Craigslist）✅ 是——接受英語市場龐大，免費刊登。瑞士最受歡迎的一般分類廣告。
Ricardo.ch拍賣／固定價格✅ 是瑞士的 eBay。二手單車受歡迎。需要註冊。
anibis.ch分類廣告✅ 是類似 tutti.ch。
Facebook Marketplace社交分類廣告✅ 是在瑞士的單車銷售非常活躍。加入本地「Verkauf」群組。

作為遊客／外國人售賣的流程：

  1. 你不需要瑞士銀行帳戶就可以在 tutti.ch 或 Facebook Marketplace 上賣車——貨到付款即可。
  2. Ricardo.ch 可能需要瑞士銀行帳戶才能收款。替代方案：現金、TWINT（需要瑞士電話／銀行）或 PayPal。
  3. 專業提示：在旅程結束前幾天刊登廣告，標明單車可在你的終點城市（如巴塞爾、沙夫豪森，甚至阿姆斯特丹）交收。
  4. 包括照片、里程，並註明你是出售保養良好的旅遊單車的遊客。

回購／Trade-In 計劃

這是最方便的選擇（如果可用的話）：

商店回購計劃備註
Decathlon Switzerland✅ 兩年滿意保證；Decathlon 單車回購如果你買了 Decathlon 單車（如 Riverside 旅遊單車），你可能可以以折扣價賣回給他們。查看 decathlon.ch 的當前政策。
Veloplus購買新車時可能提供 trade-in較不可能向非居民提供現金回購，但值得一問。
本地單車店有些提供寄售服務琉森／蘇黎世的較小店舖可能同意寄售你的單車。

典型回購價值：經過2-3星期旅程的保養良好單車，原價的40-60%。

寄售店（二手）

  • Brockenstube / Brocki — 瑞士慈善二手店（如 Heilsarmee Brocki, Brocki Zürich）接受捐贈單車。不太理想，但作為最後手段。
  • ReVelo — 蘇黎世的二手單車店，售賣翻新單車。如果你的單車狀態良好，他們可能會買。

替代方案：捐贈給慈善機構

如果賣車太麻煩：

  • 捐贈給瑞士慈善單車計劃，如 Velafrica（收集二手單車送往非洲）或當地的 Brocki
  • 留在單車店並附上便條（不太可靠）

沿途售賣

由於萊茵河路線經過許多瑞士城鎮，你也可以考慮：

  • 巴塞爾 — 進入德國前的最後瑞士城市。通過 tutti.ch / Facebook Marketplace 在那裡出售
  • 沙夫豪森／萊茵瀑布 — 靠近德國邊境，容易乘火車到蘇黎世

7. 後勤貼士

增值稅（MWST）出口退稅

  • 瑞士不屬於歐盟。如果你將單車出口到其他國家，你可以申請退還瑞士增值稅（8.1%）。
  • 但是：增值稅退稅通常適用於購買後30天內出口的貨物，且貨物需離開瑞士。你需要在瑞士邊境（如巴塞爾、日內瓦機場、沙夫豪森）向海關出示單車、原始收據和商店的免稅表格。
  • 實際情況：瑞士大多數單車店不為個人提供免稅購物。一些較大的商店（Decathlon）如果你要求可能會提供。不要依賴這個。

文件與證件

  • 無需登記——瑞士的單車不需要登記或牌照
  • 無需進口關稅——從香港帶單車進入瑞士，個人使用免稅
  • 身份證／護照——如果在商店購買，請攜帶（他們可能需要身份證辦理付款）
  • 單車序列號——購買新車時記下車架號碼（對保險／賣車有用）

保險

  • 考慮涵蓋單車被盜／損壞的旅遊保險
  • 瑞士單車被盜率中等，但並非偏低——妥善鎖好單車
  • 瑞士的家庭財產保險涵蓋單車被盜（但你是訪客）

運送／飛行

  • 從香港帶單車飛行：單車箱費用因航空公司而異（通常每次 USD 100-200）。國泰航空、瑞士航空等接受單車箱作為托運行李。
  • 在瑞士購買：你可以要求商店組裝和調校單車，安裝貨架和袋。大多數商店會這樣做。

貨架和袋

  • 瑞士出售的大多數旅遊單車（Decathlon、Veloplus 等）都配有後貨架安裝點
  • Ortlieb 袋（德國品牌）在瑞士戶外／單車店廣泛有售——非常適合單車露營
  • 後貨架 + 馬鞍袋套裝預算約 CHF 150-300

8. 總結建議

方案 A：在瑞士購買（有條件推薦）

最佳方法：

  1. 在琉森或蘇黎世的 Decathlon 網上訂購並取車（最划算）——Decathlon Riverside 520/920
  2. 或前往 Veloplus Zürich 配備合適的旅遊單車
  3. 或前往 Cycle Store Zürich/Andermatt 購買高級碎石路／探險單車（Specialized Diverge）
  4. 從蘇黎世或琉森乘火車帶新單車到 Andermatt
  5. 從 Andermatt 開始萊茵河之旅
  6. 在旅程結束時（鹿特丹或終點城市），通過 Facebook Marketplace 或 tutti.ch 出售

優點：

  • 單車適合瑞士地形（齒輪、煞車、輪胎適合阿爾卑斯山）
  • 購買前可以試騎
  • 無需支付單車箱的機場行李費
  • 可以在結束時出售——收回40-60%成本

缺點：

  • 瑞士零售價比香港高約15-25%
  • 選擇比香港有限

方案 B：從香港帶來

最佳方法：

  1. 在香港購買單車（更便宜，亞洲／全球品牌選擇更廣）
  2. 將單車裝箱飛往蘇黎世機場
  3. 在蘇黎世機場組裝（IKEA式組裝站或酒店）
  4. 從蘇黎世乘火車到 Andermatt（~2小時）
  5. 在旅程結束時，仍然嘗試在歐洲出售（鹿特丹或沿途城市）

優點：

  • 較低的購買價格（無8.1%瑞士增值稅，較低的零售加成）
  • 你熟悉在香港騎過的單車
  • 更多型號選擇

缺點：

  • 航空公司單車箱費用（每次 USD 100-200）
  • 在機場組裝／拆卸的麻煩
  • 飛行中損壞的風險
  • 在結束時較難出售（歐盟／瑞士買家可能偏好本地品牌）
  • 如果帶單車飛回家，需支付回程費用

這次行程（Rhine Cycle 2027，Andermatt 出發）的最佳建議：

➡️ 在琉森或蘇黎世的 Decathlon 購買新的 Riverside 520 或 920。這些單車性價比極高（CHF 500-1,200），是專為旅遊設計的鋼架單車，配備貨架安裝點和可靠的零件。乘火車到 Andermatt（SBB 火車允許攜帶單車，需購買單車票）。沿萊茵河騎到鹿特丹。通過 Facebook Marketplace / tutti.ch 以購買價的50-60%出售。淨成本：約 CHF 300-500——比帶自己的單車飛過來更便宜。

如果你想要更高級的體驗，Veloplus Zürich 可以為你配備 Cube Touring 或類似的單車，價格約 CHF 1,200-1,800，出售時可收回約 CHF 600-900。

避免：除非你事先確認庫存，否則不要在 Andermatt 本身購買。村裡的商店以租賃為主，銷售庫存有限。
