路線概覽

EuroVelo 15 — 萊茵河路線

起點：瑞士安德馬特 (Andermatt) — 萊茵河源頭

終點：荷蘭鹿特丹 (Rotterdam) — 北海海岸

路線長度：約 1,450 公里（官方標示 1,233 公里，含改道約 1,450 公里）

途經國家：🇨🇭 瑞士 → 🇫🇷 法國（阿爾薩斯） → 🇩🇪 德國 → 🇳🇱 荷蘭

建議行程：14–21 天，每日騎行約 70–100 公里

💡 使用電動輔助 (pedelec) 每日可輕鬆騎行 80–100+ 公里，包括上坡路段。

電動單車各國法例

歐盟統一標準（適用於德國、法國、荷蘭）

根據歐盟法規 (EU) No 168/2013，只要你的電動輔助單車符合以下條件，就視為一般單車，無需車牌、保險、頭盔，可在所有單車徑合法行駛：

  • ✅ 最大功率 ≤ 250W
  • ✅ 輔助速度上限 25 km/h
  • ✅ 必須踩踏才有輔助（非油門驅動）
  • ✅ 符合 EN 15194 歐洲安全標準
國家功率上限速度上限頭盔車牌/保險單車徑備註

⚠️ 重要提醒

若你使用 350W 套件：

  • 🇨🇭 瑞士：✅ 仍然合法（500W 以下為「輕型電單車」，16歲以上無需牌照）
  • 🇩🇪 / 🇫🇷 / 🇳🇱：❌ 理論上超標（使用高功率套件須以 S-pedelec 規格登記，包括車牌、保險、頭盔）
  • 建議：將套件設定在 250W/25km/h 模式以符合四國法例。瑞士的 500W 上限提供緩衝，但建議全程保持 250W 設定以免在歐盟國家有麻煩。

補給與物資

🛒 沿途補給點

EV15 萊茵河路線沿途經過大量城鎮，補給非常方便：

  • 瑞士段 — Andermatt、Luzern、Basel：超市（Coop、Migros）密集，每 10–20 公里有補給點
  • 德國段 — 萊茵河畔幾乎每個城鎮都有超市（REWE、Aldi、Lidl）、麵包店（Bäckerei）
  • 法國段 — 阿爾薩斯酒鄉：超市（Carrefour、Intermarché）、每週市集
  • 荷蘭段 — 全世界最密集的超市網絡（Albert Heijn、Jumbo）

💡 歐洲超市有完善的露營食物選擇：即食麵、能量棒、運動飲料、罐頭、麵包、芝士。

⚡ 電動單車充電攻略
  • 露營營地：大部分 Campingplatz 提供電源插座（歐標 Type C/F）
  • 餐廳/Café：入內飲一杯咖啡，請求借用插座充電（德國/Swiss 大部分很樂意）
  • 圖書館/遊客中心：部分德國小鎮提供公共充電設施
  • 電池續航管理：每日騎行 80km，18Ah 電池應足夠一日使用
  • 充電時間：標準充電約 4–6 小時（過夜充電即可）
💧 飲水與食物
  • 德國和瑞士自來水可直接飲用（Trinkwasser）。荷蘭水質也良好。
  • 沿途城鎮有大量飲水噴泉（尤其在德國和瑞士）。
  • 歐洲火車站和超市有便宜的瓶裝水（€0.50–1.00）。
  • 每日補給策略：早餐在營地/酒店吃 → 中午在途經城鎮買麵包、芝士、水果 → 晚餐在營地煮食或到餐廳用膳。

🔐 單車防盜建議
  1. 使用 U 型鎖（建議 Abus 或 Kryptonite，非鋼纜鎖）
  2. 睡覺時將單車帶入帳篷或房間
  3. 安裝 AirTag 或類似追蹤器隱藏在車架內
  4. 拆卸電池隨身攜帶（既防盜又方便充電）
  5. 歐洲電動單車盜竊猖獗，切勿讓單車無人看管
🌤️ 騎行安全
  • 歐洲車輛靠右行駛（與香港相同）
  • 夜間必須安裝前後燈（法律要求）
  • 德國對 ebike 改裝執法嚴格，最高可罰款 €500
  • 瑞士阿爾卑斯路段 5 月氣溫可能降至 3–10°C，需準備保暖衣物
  • 萊茵河地區潮濕多雨，使用防水袋保護電子設備
